# Creating an Account on SVM Pro

SVM Pro is a Web 3 app. It means that to connect a third party web 3 wallet is needed, suck as Metamask, Coinbase wallet, Phantom, etc. But for those who don't want to manipulate web 3 wallets, we have provided embedded wallets if you connect with email,

Go to [https://pro.sportvalue.app/](https://pro.sportvalue.app/) to see the. main dashboard.

<figure><img src="../.gitbook/assets/Screenshot 2026-03-13 at 00.30.55.png" alt=""><figcaption></figcaption></figure>

From there select 'Sign In'



<figure><img src="../.gitbook/assets/Screenshot 2026-03-13 at 00.31.07.png" alt=""><figcaption></figcaption></figure>

You can choose to sign in with email, Google or with your own web 3 wallet.
